# TexThor
TexThor

TexThor is a Text Editor program that was developed  as a University of North Sumatra assignment.
This program made by a c++ code based on Object Oriented Programming method.

FEATURES
- Create File
- Add to file
- Read File
- Copy File
- Delete File
- Empty File
- Compile & Run File
